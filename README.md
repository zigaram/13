# World of Aquariums — worldofaquariums.com

Programmatic SEO site targeting 14.3M monthly searches across the aquarium niche.
Built with Next.js 14 (App Router), Tailwind CSS, and TypeScript.

## Content Strategy

- **246 editorial articles** across 13 pillars (see content strategy spreadsheet)
- **1,000+ programmatic pages** generated from structured data
- **4 interactive calculators** (tank volume, stocking, heater size, dimensions)
- **Fish compatibility matrix** generating 500+ unique pages

## Monetization

- **Primary**: Raptive display ads (6 ad slots per page)
- **Secondary**: Amazon affiliate links on product reviews
- **Tertiary**: Direct affiliate (Bulk Reef Supply, Chewy, etc.)

## Key SEO Features

- Full JSON-LD schema (Article, Product, HowTo, FAQ, BreadcrumbList)
- Canonical URLs on every page
- Intelligent internal linking engine
- Automated sitemap generation (next-sitemap)
- Breadcrumb navigation with schema markup
- Open Graph + Twitter Card metadata
- Image optimization (AVIF/WebP via next/image)

## Development

```bash
npm install
npm run dev       # Start dev server on localhost:3000
npm run build     # Production build
npm run start     # Start production server
```

## Data Pipeline

Fish, plant, disease, and tank size data lives in `/src/data/` as JSON files.
These are generated by build scripts (to be created) that pull from:

1. Curated spreadsheets (species database)
2. FishBase API (scientific data)
3. Product database (equipment reviews)

## Deployment

Optimized for Vercel deployment with `output: 'standalone'` in next.config.js.
